ðŸ“š Flexbox Properties Quick Reference

Understanding flexbox is like understanding how items organize themselves in a container - think of it as a smart organizer that arranges objects on a shelf automatically!

Container Properties

  • display: flex - Creates flex container
  • flex-direction - Sets main axis direction
  • justify-content - Aligns along main axis
  • align-items - Aligns along cross axis
  • flex-wrap - Controls wrapping
  • gap - Spacing between items

Item Properties

  • flex-grow - How much to grow
  • flex-shrink - How much to shrink
  • flex-basis - Initial size
  • flex - Shorthand for above
  • align-self - Individual alignment
  • order - Visual order
